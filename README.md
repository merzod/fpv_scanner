# FPV Scanner
Custom FPV 5.8GHz scanner

## Hardware:
* Arduino Micro
* RX5808 FPV module (hacked for controll over SPI https://github.com/sheaivey/rx5808-pro-diversity/blob/master/docs/rx5808-spi-mod.md)
* Nokia 5110 LCD
* 4 buttons
#### Optional:
* 1 cell Lipo battery
* USB Lipo charger
* box for all the stuff
